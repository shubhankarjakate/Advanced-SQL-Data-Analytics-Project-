 Advanced-SQL-Data-Analytics-Project-

📊 SQL Data Analysis Portfolio

A collection of SQL scripts demonstrating real-world data analysis techniques using analytical queries, window functions, Common Table Expressions (CTEs), and business-oriented reporting.

This repository focuses on solving practical business problems using SQL rather than simply demonstrating SQL syntax. Each script explores a specific analytical pattern commonly used by data analysts and business intelligence professionals.



 🎯 Objectives

- Practice writing analytical SQL queries
- Strengthen problem-solving skills using SQL
- Understand business reporting techniques
- Learn advanced SQL concepts including Window Functions and CTEs
- Build a portfolio of reusable SQL analysis patterns



 📂 Repository Structure

| File                         | Analysis                                           |

| 2 performance analysis.sql   | Year-over-Year (YoY) performance analysis          |
| 3 data segmentation.sql      | Customer and product segmentation                  |
